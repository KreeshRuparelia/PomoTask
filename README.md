# Pomodoro Task Manager

PomoTask is an interactive web application that helps you focus by following the Pomodoro technique, segmenting work time into focus sessions, short breaks, and long breaks. The Pomodoro technique has been scientifically shown to boost productivity and offer mood benefits.

## How The Pomodoro Technique Works

1. **Focus**: Work for 25 minutes (or your custom duration)
2. **Short Break**: Take a 5-minute break
3. **Repeat** until you complete 4 focus sessions
4. **Long Break**: Enjoy a 15-minute break
5. **Start Again**: Begin a new cycle

## Features

### Smart Pomodoro Timer
- **Three Timer Modes**: Focus sessions, short breaks, and long breaks
- **Customizable Durations**: Adjust each phase from 1-60 minutes
- **Automatic Transitions**: Seamlessly moves between work and break phases
- **Intelligent Scheduling**: Automatically triggers long breaks after every 4 focus sessions

### Integrated Task Management
- Create and manage your task queue
- Check off completed tasks
- Delete tasks with one click
- Plan what you'll accomplish during each focus session

### Progress Tracking
- Visual progress dots showing your position in the work cycle
- Total focus sessions completed counter
- Live countdown to your next long break

### Beautiful Interface
- Clean, modern design
- Responsive two-column layout
- Smooth animations and transitions
- Easy-to-use controls

## Tech Details

- **React** - Component-based UI
- **JavaScript** - Custom Timer class with interval-based countdown
- **CSS3** - Glassmorphism effects and responsive design
- **React Hooks** - State management (useState, useRef, useEffect)
